# NAS YouTube Translator — 产品发布计划

## 目标
将产品包装成"下载即用"的形式，让用户双击 EXE 即可使用，无需安装 Python 或其他依赖。

---

## 模块一：环境验证（10分钟）

### 1.1 验证 PyInstaller 配置
- [ ] 检查 `YTSubViewer.spec` 文件内容
- [ ] 确认包含所有必要文件（前端、配置、模型）
- [ ] 检查 hidden imports 是否完整

### 1.2 验证依赖完整性
- [ ] 检查 `requirements.txt` 是否包含所有运行时依赖
- [ ] 检查 `requirements-build.txt`（PyInstaller 依赖）
- [ ] 确认 faster-whisper 模型路径配置

### 1.3 验证外部工具
- [ ] 确认 ffmpeg 路径配置正确
- [ ] 确认 mpv 路径配置正确
- [ ] 检查 `.tools/` 目录结构

## 风险与应对

| 风险 | 影响 | 应对方案 |
|------|------|----------|
| PyInstaller 构建失败 | 高 | 检查 hidden imports，修复配置 |
| EXE 启动崩溃 | 高 | 检查依赖，添加错误日志 |
| 包体过大 | 中 | 优化依赖，压缩资源 |
| 用户无法运行 | 中 | 提供详细使用说明 |
